# Zorvyn Finance Dashboard

A secure, scalable finance data processing and access control system built with **Node.js, Express, MySQL (Aiven), Kafka (Aiven), and React**.

## Features
- **Role-Based Access Control (RBAC)**: Viewer, Analyst, and Admin roles.
- **Financial Record Management**: Full CRUD operations for income/expense entries.
- **Dashboard Summary**: Real-time analytics and trend visualizations.
- **Event-Driven Architecture**: Kafka-integrated audit logging and activity streams.
- **Premium UI**: Modern, dark-themed dashboard with fluid interactions.

## Project Structure
- `backend/`: Express.js API with modular architecture.
- `frontend/`: React + Vite with a high-fidelity design system.
- `shared/`: Common types and utility functions.
